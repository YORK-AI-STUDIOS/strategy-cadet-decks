# strategy-cadet-decks

Hosting for Strategy Cadet HTML presentations. Testing phase — shared with collaborators for review.

## URLs

- Landing page: https://york-ai-studios.github.io/strategy-cadet-decks/
- Each deck: https://york-ai-studios.github.io/strategy-cadet-decks/<deck-name>/

## Adding a deck

1. Create a folder `<deck-name>/` in the repo root
2. Put the presentation HTML in it as `index.html` (plus any assets)
3. Commit and push
4. GitHub Pages publishes it within ~1 minute at `/<deck-name>/`

## Notes

- Repo is **public** — only dummy/sanitised test content goes here. No real client or prospect names.
- Pages serves from the `main` branch root.
- When this moves to client-facing work, revisit: private repo + paid plan, or a custom domain.
